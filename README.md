# Husna Siddiqa — Portfolio Website

Personal portfolio website for **Husna Siddiqa**, AI/ML Engineer & Technical Team Lead at Setuserv Informatics (Gilead Sciences).

🔗 **Live site:** [husnasiddiqa.github.io/PortFolio_Website](https://husnasiddiqa.github.io/PortFolio_Website)

---

## About

A fully responsive, single-page portfolio built with vanilla HTML, CSS, and JavaScript. Designed to showcase professional experience in GenAI platform engineering, multi-agent orchestration, and enterprise analytics for pharmaceutical clients.

---

## Sections

- **Hero** — Introduction with animated typing effect and social links
- **About** — Professional bio and key impact stats
- **Skills** — Categorized technical skills (GenAI/LLM, Analytics Engineering, ML/DL, Cloud & Data, Languages, Tools)
- **Experience** — Work history with detailed bullet points (Setuserv / Gilead Sciences)
- **Projects** — C3PO multi-agent platform, Databricks AI/BI Genie, Image Captioning, Pneumonia Detection
- **Contact** — Contact form and direct links

---

## Tech Stack

- HTML5, CSS3, JavaScript (vanilla — no frameworks)
- Google Fonts (Inter, Fira Code)
- Font Awesome 6 icons
- Google Analytics 4 (event tracking for resume downloads and social link clicks)

---

## Features

- Dark navy + teal theme
- Smooth scroll with active nav link highlighting
- Intersection Observer fade-in animations
- Typing animation in hero section
- GA4 custom event tracking (resume downloads, LinkedIn, GitHub, LeetCode, email clicks)
- Fully responsive (mobile, tablet, desktop)

---

## Setup

1. Clone the repo
2. Replace `G-XXXXXXXXXX` in `index.html` with your GA4 Measurement ID
3. Update project GitHub links in `index.html` with actual repo URLs
4. Add `Husna_Siddiqa.pdf` to the root folder for the resume download button
5. Open `index.html` in a browser
